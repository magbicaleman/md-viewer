# MD Viewer

This project opens a single Markdown file or scans a folder tree for Markdown notes.

## Quick Paths

- Open the `examples` folder to see recursive indexing.
- Open this file directly to test single-file mode.
- Follow the internal link to [Daily Note](./notes/daily.md).
- Visit [Google](https://google.com) to test an external website link.

## Reader Controls

- Switch theme
- Adjust type size
- Adjust reading width

## Refresh Check

If you can read this section, the viewer picked up a later file edit.
